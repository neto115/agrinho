# # AgroForte - Futuro Sustentável 🌿

Uma landing page moderna, responsiva e interativa desenvolvida para promover práticas agrícolas sustentáveis unidas à tecnologia de ponta (Agrotech).

## 🚀 Funcionalidades

- **Design Responsivo:** Adaptável para dispositivos móveis, tablets e desktops.
- **Seção de Pilares:** Apresentação clara dos focos do projeto (Tecnologia, Gestão Hídrica e Cuidado com o Solo).
- **Simulador Interativo (JavaScript):** Permite ao usuário simular a economia anual de água e redução de emissão de CO₂ com base no tamanho da sua propriedade em hectares.
- **Formulário de Contato:** Validação simples de envio para captação de leads interessados em agricultura sustentável.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica da página.
- **CSS3:** Estilização personalizada, uso de variáveis CSS, Flexbox, e animações sutis.
- **JavaScript (ES6):** Manipulação do DOM para cálculo em tempo real e interatividade do formulário.
- **Font Awesome:** Ícones vetoriais modernos.
- **Gemini Utilizado:** Para otimização de codigo.

## 📁 Como Executar o Projeto

1. Baixe os arquivos ou clone este repositório.
2. Certifique-se de que os arquivos `index.html`, `style.css` e `script.js` estejam na **mesma pasta**.
3. Dê um duplo clique no arquivo `index.html` para abri-lo diretamente no seu navegador de preferência.
